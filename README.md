## 1. 快速开始

### 1.1 下载应用

请访问 [Github仓库](https://github.com/Mofan2020/SmartNote/releases/) 下载本程序的ZIP压缩包，并解压。**建议下载最新的稳定版本。**

### 1.2 启动应用

在 Finder 中找到 `SmartNote.app` 并双击打开，或在 Launchpad 中点击图标。

## 2. 资料管理

### 2.1 导入资料

#### 方式一：导入文件
1. 点击工具栏「添加」按钮
2. 选择「导入文件」
3. 在文件选择器中选择要导入的文件
4. 选择拷贝文件到资料库（推荐）或仅创建关联

（支持格式：PDF、Word、PPT、图片、文本）

#### 方式二：手动新建
1. 点击工具栏「添加」按钮
2. 选择「新建资料」
3. 填写资料名称、选择分类、输入内容
4. 可选添加考点关键词

### 2.2 查看资料

1. 在侧边栏选择分类或在资料列表中点击资料
2. 右侧显示资料详情
3. 可查看文档内容、关键词、文件信息

### 2.3 编辑资料

- **实时编辑**：在资料详情页面直接编辑内容，自动保存
- **编辑分类**：点击「编辑分类」按钮选择新分类
- **编辑关键词**：点击关键词旁「编辑」按钮输入新关键词

### 2.4 导出资料

在资料详情页面，点击「导出」菜单：
- 导出为 PDF
- 导出为文本
- 导出为 Markdown
- 复制内容

### 2.5 OCR 文字识别

对于图片格式的资料：
1. 打开资料详情
2. 点击「文字识别」按钮
3. 系统自动提取图片中的文字

## 3. AI 功能

### 3.1 配置 AI

1. 打开「设置」（应用菜单 → 设置或 `Cmd+,`）
2. 进入「AI 分析」标签页
3. 配置选项：

| 选项 | 说明 |
|------|------|
| 启用 AI | 开关 AI 功能 |
| AI 提供商 | LM Studio / OpenAI / Anthropic |
| 服务器地址 | 例如 `http://localhost:1234` |
| Token | 认证 Token（可选） |
| 模型 ID | 要使用的模型名称 |
| Temperature | 生成随机性 (0-1) |
| Max Tokens | 最大生成长度 |

### 3.2 考点提取

1. 侧边栏选择「考点提取」
2. 在左侧选择一份资料
3. 系统自动提取关键词
4. 可点击「AI 分析」获取详细分析

### 3.3 AI 对话

1. 侧边栏选择「AI 对话」
2. 在底部输入框提问
3. AI 会根据上下文回答

### 3.4 智能阅卷

1. 侧边栏选择「智能阅卷」
2. 上传文件或选择资料
3. 标记用户回答、原题、标准答案
4. AI 自动批改并给出详细分析

## 4. 复习计划

### 4.1 创建计划

1. 侧边栏选择「复习计划」
2. 点击「新建」→「新建计划」
3. 填写考试科目、考试日期、复习要点
4. 点击「创建」

### 4.2 管理任务

- 查看今日任务 - 主页显示今日任务数量
- 标记完成 - 点击任务左侧圆圈
- 查看进度 - 选择计划查看详细进度
- 按日浏览 - 点击日历按钮切换日期

### 4.3 联动日历

创建计划时，可选择同步到 macOS 日历，自动创建复习提醒。

## 5. 番茄钟

专注学习计时器：

1. 侧边栏选择「番茄钟」
2. 设置专注时长（15-90分钟）
3. 点击「开始专注」
4. 计时结束后自动休息

**功能特点：**
- 自定义工作/休息时长
- 学习数据统计
- 科目分布图表
- 系统通知提醒

## 6. 错题本与背诵卡片

### 6.1 错题本

1. 侧边栏选择「错题本」
2. 点击「添加」录入错题
3. 支持手动录入、拍照识别
4. 按艾宾浩斯遗忘曲线自动推送复习

### 6.2 背诵卡片

1. 侧边栏选择「背诵卡片」
2. 点击「添加」创建卡片
3. 或从资料自动生成卡片
4. 复习时点击卡片翻转背诵

## 7. 考试倒计时

1. 侧边栏选择「考试倒计时」
2. 点击「添加考试」
3. 填写考试名称、日期、科目
4. 首页常驻显示倒计时

**功能：**
- 自动计算剩余天数
- 临近考试标红提醒
- 过期考试自动归档

---

## 8. PDF 批注

1. 打开 PDF 资料
2. 使用工具栏选择批注工具
3. 支持：高亮、下划线、删除线、方框、文本备注

**功能：**
- 6种批注工具 + 6种颜色
- 批注自动保存
- 导出为文本/Markdown/JSON

## 9. 重复文件清理

1. 侧边栏选择「重复清理」
2. 选择要扫描的文件夹
3. 系统自动识别重复文件
4. 选择保留策略（一键清理）

**支持格式：** PDF、DOC、DOCX、TXT、MD、PPT、PPTX

## 10. P2P 社交（本功能停止维护）

端到端加密的 P2P 社交功能：

### 10.1 创建身份

1. 侧边栏选择「P2P 社交」
2. 创建或复用现有身份
3. 自动生成 RSA-2048 密钥对

### 10.2 添加好友

1. 点击「添加好友」
2. 输入对方 IPv6 地址和端口
3. 等待对方同意连接

### 10.3 功能特点

- 纯 IPv6 P2P 直连
- 端到端加密（RSA-2048 + AES-256）
- 加密文字聊天
- IP 黑名单
- 后台常驻开关

## 13. 白板

### 13.1 创建白板

- 每次打开时，将默认创建新的白板
- 系统会自动保存您在白板上的所有编辑，在右下角可以查看状态与缩放比例
- 您可以打开或管理过往的白板

### 13.2 绘制图形

- 白板功能内置了圆形、三角形、矩形工具，您可以快速开始绘制
- 您可以通过UI界面快速修改颜色与线条粗细等参数

### 13.3 画笔与橡皮工具

- 您可以通过画笔工具快速绘制任意内容，并修改颜色与线条粗细
- 通过橡皮擦，您可以快速擦除绘制的内容

### 13.4 撤销与重做

- 通过**快捷键**（请在菜单栏中的**编辑**中查看）或**菜单栏/编辑**中的**撤销**与**重做**，您可以快速撤销或重做您的操作
