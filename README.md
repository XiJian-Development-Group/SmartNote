## 1. 快速开始

### 1.1 下载应用

请访问 <a href="https://github.com/Mofan2020/SmartNote/releases/">Github仓库</a> 下载本程序的ZIP压缩包，并解压。**注意：建议您下载最新版本的程序，以获得更好的使用体验。**

### 1.2 启动应用

在 Finder 中找到 `SmartNote.app` 并双击打开，或在 Launchpad 中点击图标。

## 2. 资料管理

### 2.1 导入资料

#### 方式一：导入文件
1. 点击工具栏「添加」按钮
2. 选择「导入文件」
3. 在文件选择器中选择要导入的文件
4. 支持格式：PDF、Word、PPT、图片、文本

#### 方式二：手动新建
1. 点击工具栏「添加」按钮
2. 选择「新建资料」
3. 填写资料名称、选择分类、输入内容
4. 可选添加考点关键词

### 2.2 查看资料

1. 在侧边栏选择分类或在资料列表中点击资料
2. 右侧显示资料详情
3. 可查看文档内容、关键词、文件信息

### 2.3 编辑资料

#### 实时编辑
- 在资料详情页面，直接编辑「文档内容」区域
- 内容会自动保存（显示「自动保存」标识）

#### 编辑分类
1. 在资料详情中点击「编辑分类」按钮
2. 选择新的分类（课件/真题/笔记/其他）
3. 点击「保存」

#### 编辑关键词
1. 在资料详情中点击关键词旁的「编辑」按钮
2. 用逗号分隔输入关键词
3. 点击「保存」

### 2.4 导出资料

在资料详情页面，点击「导出」菜单：
- **导出为 PDF** - 生成 PDF 文档
- **导出为文本** - 保存为 .txt 文件
- **复制内容** - 复制到剪贴板
- **打开原文件** - 用默认应用打开

### 2.5 其他操作

- **收藏资料** - 右键点击资料选择「收藏」
- **删除资料** - 右键点击资料选择「删除」
- **搜索资料** - 在顶部搜索框输入关键词

## 3. AI 功能

### 3.1 配置 AI

1. 打开「设置」（应用菜单 → 设置或快捷键 `Cmd+,`）
2. 进入「AI 分析」标签页
3. 配置选项：

| 选项 | 说明 |
|------|------|
| 启用 AI | 开关 AI 功能 |
| AI 提供商 | LM Studio / OpenAI / Anthropic |
| 服务器地址 | 例如 `http://localhost:1234` |
| Token | 认证 Token（可选） |
| 模型 ID | 要使用的模型名称 |
| Temperature | 生成随机性 (0-1) |
| Max Tokens | 最大生成长度 |

#### 推荐配置 (LM Studio)

```
服务器地址: http://localhost:1234
模型 ID: llama-3-8b-instruct
```

### 3.2 考点提取

1. 侧边栏选择「考点提取」
2. 在左侧选择一份资料
3. 系统自动提取关键词
4. 可点击「AI 分析」获取更详细的分析

#### AI 分析类型

- **核心考点** - AI 分析文档提取重点
- **内容总结** - AI 生成内容摘要
- **复习题目** - AI 生成练习题

#### 保存分析结果

分析完成后，点击「保存 PDF」即可导出。

### 3.3 AI 对话

1. 侧边栏选择「AI 对话」
2. 在底部输入框提问
3. AI 会根据上下文回答

### 3.4 OCR 文字识别

对于图片格式的资料：
1. 打开资料详情
2. 点击「文字识别」按钮
3. 系统自动提取图片中的文字

## 4. 复习计划

### 4.1 创建计划

#### 方式一：新建计划
1. 侧边栏选择「复习计划」
2. 点击「新建」→「新建计划」
3. 填写考试科目、考试日期、复习要点
4. 点击「创建」

#### 方式二：从资料导入
1. 侧边栏选择「复习计划」
2. 点击「新建」→「从资料导入知识点」
3. 选择已提取知识点的资料
4. 勾选要导入的知识点
5. 创建计划

### 4.2 管理任务

- **查看今日任务** - 主页显示今日任务数量
- **标记完成** - 点击任务左侧圆圈
- **查看进度** - 选择计划查看详细进度
- **按日浏览** - 点击日历按钮切换日期
- **删除计划** - 右键点击计划选择「删除」

### 4.3 联动日历

创建计划时，可选择同步到 macOS 日历：
- 自动创建复习提醒
- 定时推送通知

## 常见问题

### Q: 导入的 PDF 没有文字内容？
A: PDF 需要是可搜索的文本型 PDF。扫描的图片型 PDF 需要先使用 OCR 功能识别。

### Q: AI 连接失败？
A: 
1. 确认 LM Studio 已启动并加载模型
2. 检查服务器地址和端口是否正确
3. 确认模型 ID 正确

### Q: 如何备份资料？
A: 资料保存在 `~/Library/Application Support/SmartNote/` 目录，可以直接复制备份。
