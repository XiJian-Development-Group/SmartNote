## 1. 快速开始

### 1.1 下载应用

请访问 [GitHub Releases](https://github.com/XiJian-Development-Group/SmartNote/releases) 下载最新版本的 ZIP 压缩包并解压。

### 1.2 启动应用

在 Finder 中找到 `SmartNote.app` 并双击打开，或在 Launchpad 中点击图标。

## 2. 资料库

### 2.1 导入资料

#### 方式一：导入文件
1. 点击工具栏「添加」按钮
2. 选择「导入文件」
3. 在文件选择器中选择要导入的文件
4. 选择拷贝文件到资料库（推荐）或仅创建关联

（支持格式：PDF、Word、PPT、图片、文本）

#### 方式二：手动新建
1. 点击工具栏「添加」按钮
2. 选择「新建资料」
3. 填写资料名称、选择分类、输入内容
4. 可选添加考点关键词

### 2.2 分类查看

侧边栏「资料库」区域按分类展示：
- **全部资料** — 所有资料
- **课件** — 课程讲义类
- **真题** — 考试真题类
- **笔记** — 个人笔记类
- **收藏** — 星标收藏的资料

### 2.3 编辑资料

- 在资料详情页面直接编辑内容，自动保存
- 编辑分类或关键词

### 2.4 导出资料

在资料详情页面，点击「导出」菜单：
- 导出为 PDF / 文本 / Markdown
- 复制内容

### 2.5 OCR 文字识别

对于图片格式的资料，打开详情后点击「文字识别」，系统自动提取图片中的文字。

## 3. 学习工具

### 3.1 考点提取

1. 侧边栏选择「考点提取」
2. 选择一份资料
3. 系统自动提取关键词
4. 可点击「AI 分析」获取详细分析

### 3.2 AI 对话

1. 侧边栏选择「AI 对话」
2. 在底部输入框提问
3. AI 根据上下文回答

### 3.3 智能阅卷

1. 侧边栏选择「智能阅卷」
2. 上传文件或选择资料
3. 标记用户回答、原题、标准答案
4. AI 自动批改并给出详细分析

---

## 4. 计划管理

### 4.1 复习计划

1. 侧边栏选择「复习计划」
2. 点击「新建」→「新建计划」
3. 填写考试科目、考试日期、复习要点
4. 点击「创建」
5. 可按日浏览进度，标记任务完成

创建计划时可同步到 macOS 日历，自动创建复习提醒。

### 4.2 待办清单

1. 侧边栏选择「待办清单」
2. 添加、编辑、完成待办事项
3. 支持分类管理与统计

### 4.3 习惯养成打卡

1. 侧边栏选择「习惯养成打卡」
2. 创建要养成的习惯
3. 每日打卡记录进度

### 4.4 学习统计

1. 侧边栏选择「学习统计」
2. 查看学习时长、科目分布等统计数据

## 5. 实用工具

### 5.1 日记

1. 侧边栏选择「日记」
2. 新建或编辑日记条目
3. 支持分类管理与搜索
4. 内容加密保护隐私

### 5.2 PDF 批注

1. 打开 PDF 资料
2. 使用工具栏选择批注工具
3. 支持：高亮、下划线、删除线、方框、文本备注
4. 6 种批注工具 + 6 种颜色，批注自动保存
5. 可导出为文本 / Markdown / JSON

### 5.3 重复文件清理

1. 侧边栏选择「重复清理」
2. 选择要扫描的文件夹
3. 系统自动识别重复文件
4. 选择保留策略一键清理

支持格式：PDF、DOC、DOCX、TXT、MD、PPT、PPTX

## 6. AI 功能

### 6.1 配置 AI

打开「设置」（`Cmd+,`）→「AI 分析」标签页：

| 选项 | 说明 |
|------|------|
| 启用 AI | 开关 AI 功能 |
| AI 提供商 | LM Studio / OpenAI / Anthropic |
| 服务器地址 | 例如 `http://localhost:1234` |
| Token | 认证 Token（可选） |
| 模型 ID | 要使用的模型名称 |
| Temperature | 生成随机性 (0-1) |
| Max Tokens | 最大生成长度 |

### 6.2 考点提取

在考点提取页面选择资料，系统自动提取关键词，可点击「AI 分析」获取详细分析。

### 6.3 AI 对话

在 AI 对话页面输入问题，AI 根据上下文回答。

### 6.4 智能阅卷

上传文件或选择资料，标记用户回答与标准答案，AI 自动批改并给出分析。

## 7. 番茄钟

专注学习计时器：

1. 侧边栏选择「番茄钟」
2. 设置专注时长（15-90 分钟）
3. 点击「开始专注」
4. 计时结束后自动休息

**功能：**
- 自定义工作 / 休息时长
- 学习数据统计
- 科目分布图表
- 系统通知提醒

## 16. 自动更新

应用内置自动更新功能，基于 GitHub Releases：

- **检查更新**：手动或定时检查 GitHub 仓库的最新 Release
- **自动下载**：启用后自动下载更新包
- **自动安装**：下载完成后自动解压并安装到 `/Applications`
- **版本比较**：自动跳过与当前版本相同的 Release
- **更新渠道**：可选择 Latest（稳定版）或 Pre-release（预览版）
- **自定义仓库**：支持修改 GitHub 仓库地址，可用于自建分发
- **更新日志**：在设置中查看下载和安装日志

在「设置 → 通用 → 更新」中配置：

| 选项 | 说明 |
|------|------|
| 自动下载并安装更新 | 开关自动更新 |
| 检查间隔 | 1–168 小时 |
| 仓库地址 | 自定义 Owner/Repo |
| 更新渠道 | Latest / Pre-release |
